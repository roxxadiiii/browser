# directory for the policies of chromium based browsers

## chromium
```
sudo mkdir -p /etc/chromium/policies/managed
```

## for brave
```
sudo mkdir -p /etc/brave/policies/managed
```

## for ungoogled-chromium
```
sudo mkdir -p /etc/chrmoium/policies/managed
# same as chromium
```

# for helium-browser
```
sudo mkdir -p /etc/chromium/policies/managed
# same as chromium
```

# for thorium-browser

```
sudo mkdir -p /etc/chromium/policies/
# same as chromium 
```
